# 🚀 DevSphere – Developer Portfolio & Chat Dashboard

DevSphere is a sleek, full-stack developer portfolio dashboard featuring an interactive user interface and a real-time messaging system.

---

## 🌐 Live Links

| Component | Deployment URL |
| :--- | :--- |
| **Frontend** | [dev-sphere-three.vercel.app](https://dev-sphere-three.vercel.app) |
| **Backend API** | [devsphere-8dz9.onrender.com](https://devsphere-8dz9.onrender.com) |

---

## ✨ Features

* **💼 Modern Portfolio UI:** Clean, professional aesthetic tailored for developers.
* **⚡ Dynamic Typing Effect:** Animated role showcasing to grab attention.
* **💬 Real-Time Chat:** Fully functional chat interface powered by a custom REST API.
* **🔗 Seamless Integration:** Connected frontend and backend communication using the Fetch API.
* **🎨 Responsive Design:** Built using Bootstrap and custom CSS for flawless viewing across all devices.
* **🌍 Cloud Deployed:** Live and operational on Vercel (Frontend) and Render (Backend).
